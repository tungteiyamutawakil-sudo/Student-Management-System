# Student Management System

## Overview
This is a desktop-based Student Management System developed using JavaFX and Maven.  
The system allows administrators and lecturers to manage student records efficiently.

---

## Features
- User Login
- Admin and Lecturer Panels
- Add, Edit and Delete Student Records
- Dashboard with GPA statistics
- Report Generation
- CSV Export

---

## Technologies Used
- Java
- JavaFX
- Maven
- MySQL

---

## How to Run
1. Open the project in IntelliJ IDEA.
2. Allow Maven dependencies to load.
3. Run the main application file.
